SocialBadge

Native iOS app that is intended to meet new people around (in 50 meters radius). It was developed for the SocialBadge company, founded by me and my university friend. The app includes registration using Facebook, observing people nearby using iBeacons, sending and receiving push notification, integration with several social media services.

  • AFNetworking to implement HTTP requests support
  • Facebook SDK (SSO integration)
  • iBeacons support
  • OAuth-like login integration for LinkedIn, Instagram, Twitter, LastFM
  • CocoaPods for project dependencies

Tanks Game

Simple game about tanks. You start from the top-left corner (green tank) and your goal is to reach the bottom-right corner without being captured by the enemies (red tanks).

  • Written in pure C++ using gedit text editor only
  • SFML is used for OpenGL drawing and key events handling
  • Self-written autogenerated tile map
  • Self-written embedded map editor
  • A* for pathfinding
